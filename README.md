# MiniProjects-Python-Fresher

This repository contains 3 mini Python projects demonstrating basic Python skills for a fresher portfolio.

## Projects

1. **calculator.py** – Mini Calculator (Python basics, loops, functions, conditionals)
2. **csv_automation.py** – CSV file handling and automation (reading, updating, writing CSV)
3. **web_scraper.py** – Web scraping using BeautifulSoup (extract quotes from website)

## Skills Demonstrated
- Python basics (loops, functions, conditionals)
- File handling (CSV)
- Web scraping (BeautifulSoup)
- GitHub repo usage for portfolio
